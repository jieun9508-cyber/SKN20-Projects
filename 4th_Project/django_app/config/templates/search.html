선택된 지역

정책 검색 이용 가이드다냥! 🐱

후배냥! 여기서는 내가 청년 정책들을 싹~ 정리해둔 데이터베이스에서 직접 검색할 수 있다냥~

🔍 Step 1: 검색어 입력하기

상단 검색창에 찾고 싶은 정책 관련 키워드를 입력해라냥!

💬 예시

  • "월세 지원" → 월세 관련 정책들이 쫙~ 나온다냥!
  • "취업 지원금" → 취업 준비생을 위한 지원금 정책이 뜬다냥!
  • "창업" → 창업 관련 모든 정책을 볼 수 있다냥!

검색어 입력 후 🔍 버튼을 누르거나 Enter 키를 치면 검색이 실행된다냥~

🎯 Step 2: 상세검색 필터 활용하기

"상세검색 열기" 버튼을 누르면 후배냥 조건에 맞는 정책만 골라낼 수 있다냥!

📋 필터 항목 설명

  • 지역: 버튼을 누르면 시/도 → 시/군/구 순서로 선택! 여러 지역 동시 선택 가능하다냥~
  • 혼인여부: 미혼/기혼 중 선택해라냥
  • 연령: 만 나이를 숫자로 입력! (예: 25)
  • 연소득: 연간 소득 범위를 만원 단위로 입력! (예: 2000 ~ 4000)
  • 학력: 고졸 미만, 대학 재학, 대학 졸업 등 선택!
  • 전공요건: 특정 전공이 필요한 정책을 찾을 때 사용!
  • 취업상태: 재직자, 미취업자, 프리랜서 등 현재 상태 선택!
  • 특화분야: 여성, 장애인, 한부모가정 등 특별 대상 정책 검색!

💬 예시: 서울 사는 26세 미취업 대졸자

지역 → 서울 / 연령 → 26 / 학력 → 대학 졸업 / 취업상태 → 미취업자 선택 후 검색!

→ 서울에서 26세 미취업 대졸자가 신청 가능한 정책들만 쏙쏙 나온다냥!

📋 Step 3: 검색 결과 확인하기

검색하면 정책 카드들이 쫙~ 나온다냥! 각 카드에는 이런 정보가 있다냥:

  • 🟢 진행중: 지금 바로 신청 가능!
  • 🟡 마감임박: 7일 이내 마감, 서둘러라냥!
  • 🔴 마감: 이미 끝났다냥ㅠ

정책명, 설명, 신청기간, 키워드 태그도 함께 표시된다냥~

🔎 Step 4: 정책 상세보기

궁금한 정책이 있으면 "자세히보기" 버튼을 눌러라냥!

  • 📋 정책 설명 (어떤 지원인지)
  • 💰 지원 정보 (지원 금액, 신청 방법, 제출 서류)
  • ✅ 자격 요건 (나이, 소득, 학력 조건)
  • 🏢 운영 기관 (담당 기관 정보)
  • 🔗 참고 URL (공식 사이트 링크)

💡 선배냥의 꿀팁!

  1. "마감제외" 체크: 진행 중인 정책만 보고 싶을 때!
  2. 초기화 버튼 (↻): 필터 설정 한 번에 리셋!
  3. 여러 지역 선택: 서울+경기 동시 검색 가능!
  4. 필터 조합 팁: 너무 많이 선택하면 결과가 적을 수 있다냥~

자, 이제 후배냥에게 딱 맞는 정책을 찾아봐라냥! 💪😺
