## 📅 주요 업데이트 내용 (2026-02-07)

1.  **프로젝트 명칭 변경**: `AI-GYM` → **`AI-ARCADE`** 로 브랜드 명이 변경되었습니다.
2.  **데이터베이스 모델 대규모 리팩토링**:
    *   `UserProfile`: `id` 필드가 `BigAutoField`로 변경되어 보다 안정적인 PK 구조를 가집니다.
    *   `UserDetail`: `job_role`, `interests` 필드가 `TextField`에서 **`JSONField`**로 변경되었습니다. (관심 분야/직군 다중 선택 지원)

4.  **UI/UX 개선**: 가독성이 향상된 게임형 UI 구성 요소들이 추가되었습니다.

## 🛠️ 로컬 환경 동기화 절차

팀원분들께서는 아래 명령어를 순서대로 실행하여 로컬 환경을 업데이트해 주세요.

### 1. 최신 코드 가져오기
```bash
git checkout main
git pull origin main
```

### 2. Backend 데이터베이스 동기화
모델 구조가 크게 변경되었으므로, 기존 DB를 초기화하거나 최신 마이그레이션을 반드시 적용해야 합니다.
```bash
cd backend
# 가상환경 활성화 상태에서 실행
python manage.py makemigrations
python manage.py migrate
```
> **[TIP]** 만약 마이그레이션 충돌이 발생할 경우, 로컬 `db.sqlite3` 파일을 삭제한 후 다시 `migrate`를 진행해 주세요. (테스트 데이터는 새로 생성 필요)

### 3. Frontend 의존성 및 빌드 확인
새로 추가된 라이브러리(Lucide-Vue 등)가 있을 수 있으므로 의존성을 재설치합니다.
```bash
cd frontend
npm install
npm run dev
```
